消息相关
  • 好友消息 #开启/关闭 + 好友消息通知 {{@privateMessage}}
    好友私聊消息，以消息的形式通知主人
  • 群消息 #开启/关闭 + 群消息通知 {{@groupMessage}}
    所有群聊的消息转发给主人，支持群聊单独设置
  • 群临时消息 #开启/关闭 + 群临时消息通知 {{@grouptemporaryMessage}}
    群员通过群发送的临时消息转发给主人
  • 群撤回 #开启/关闭 + 群撤回通知 {{@groupRecall}}
    群成员撤回的群聊消息转发给主人
  • 好友撤回 #开启/关闭 + 好友撤回通知 {{@PrivateRecall}}
    好友撤回的私聊消息转发给主人

系统设置
  • 全部通知 #开启/关闭 + 全部消息通知 {{@noticeAll}}
    设置所有通知
  • 发送给全部主人 #开启/关闭 + 通知发送给全部主人 {{@notificationsAll}}
    将上述开启的通知消息发送给全部主人，关闭则只发给第一个主人
  • 作为默认状态 #开启/关闭 + 默认状态 {{@state}}
    开启后将此插件状态功能作为默认状态功能
  • 状态定时器 #开启/关闭 + 状态任务 {{@statusTask}}

  • 设置删除缓存时间 #设置删除缓存时间300
    {{@deltime}}
    设置删除撤回缓存的时间，不建议太久 单位"秒"
  • 渲染精度 #设置渲染精度100
    {{@renderScale}}
    可选值50~200，建议100。设置高精度会提高图片的精细度，但因图片较大可能会影响发送速度
